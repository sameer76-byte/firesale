# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Fire Sale, please **do not** open a public issue.

Email the project maintainers at: **sameerprasad981@gmail.com**.

Please include:
- A description of the issue
- Steps to reproduce
- Potential impact

We will respond within 3 business days and work on a fix.

## Supported Versions

Only the latest commit on the `main` branch receives security updates.
