# Contributing to Fire Sale

Thank you for considering contributing!

## How to Report a Bug
- Open an issue with label `bug`
- Describe steps to reproduce, expected vs actual behavior
- Include screenshots if possible

## How to Suggest a Feature
- Open an issue with label `enhancement`
- Explain the feature and why it's useful

## Pull Request Process
1. Fork the repo and create a new branch
2. Make your changes (keep code style consistent)
3. Test locally with XAMPP
4. Submit a pull request describing your changes

## Code Style
- PHP: follow PSR-12 where practical
- HTML/CSS/JS: use 2-space indentation

Thank you for contributing!
